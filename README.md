# GynoApp

Una aplicación móvil diseñada para ayudar a ginecólogos a organizar y proteger la información de sus pacientes.

## ¿Qué problema resuelve?

Llevar el control de consultas, fotos y documentos médicos en el consultorio puede ser complicado. GynoApp te permite tener toda esa información en tu teléfono o tablet, organizada y protegida.

## ¿Qué puedes hacer con la app?

- **Registrar pacientes** con sus datos básicos (nombre, edad, teléfono, dirección)
- **Llevar el historial de consultas** de cada paciente (motivo, diagnóstico, tratamiento, notas)
- **Tomar fotos** durante las consultas y guardarlas protegidas (cifradas) directamente en el dispositivo
- **Buscar pacientes** rápidamente por nombre
- **Proteger el acceso** con un PIN y tu huella digital o reconocimiento facial
- **Exportar la información** como respaldo

## Privacidad y seguridad

Todo se almacena localmente en el dispositivo. Las fotos se guardan cifradas. No se envía información a servidores externos.

## Hecha para el día a día

La interfaz está diseñada pensando en un uso rápido durante la consulta: pocos toques para encontrar a una paciente, agregar una consulta o tomar una foto.
